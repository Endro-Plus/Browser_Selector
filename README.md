# Browser_Selector
Opens a window to allow you to select browsers already installed on your computer. Simply point out the .exe of the browser to the program so it knows what to open. This is great for keeping the taskbar clean!

This is intended to run on windows systems with python installed. For whatever reason, windows doesn't allow scripts to be pinned to the taskbar. If you're having this problem, check the first answer on this link: https://learn.microsoft.com/en-my/answers/questions/5753818/how-do-i-pin-a-script-file-to-the-task-bar

I'm 80% sure this would run on a mac too, but I don't have one, so I can't be sure.
